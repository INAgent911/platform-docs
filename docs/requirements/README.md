# Platform PRD Suite

This folder contains the full PRD set for the MSP platform, organized by domain and lifecycle stage.

Source inputs used:
- [Deep Research Report](/C:/source/platform-docs/docs/requirements/deep-research-report.md)
- [PRD Input Contract](/C:/source/platform-docs/docs/architecture/contracts/prd-input-contract.md)
- Architecture set under `/docs/architecture`

## Structure

- [00-governance/catalog.md](/C:/source/platform-docs/docs/requirements/00-governance/catalog.md)
- `01-foundation/` core platform PRDs
- `02-lifecycle/` customer lifecycle PRDs
- `03-experience-ai/` UX, AI, and usage transparency PRDs

## Usage

1. Start with the PRD catalog.
2. Approve or edit each PRD's open questions.
3. Freeze approved PRDs and feed them to Codex by phase.
